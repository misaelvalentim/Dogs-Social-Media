# Dogs-Social-Media

## Get started

### Installation

- git clone https://github.com/misaelvalentim/Dogs-Social-Media.git
- cd dogs
- npm install
- npm run dev

##### You're all set! The app should open automatically in your browser, usually at http://localhost:3000/
